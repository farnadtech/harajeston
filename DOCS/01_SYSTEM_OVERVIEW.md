# نمای کلی سیستم

## نام پروژه
**سامانه حراجی آنلاین ایرانی**

## توضیحات
پلتفرم جامع حراجی آنلاین با پشتیبانی کامل از زبان فارسی، تقویم شمسی، و ویژگی‌های بومی‌سازی شده برای بازار ایران.

## ویژگی‌های کلیدی
1. حراجی زنده با تایمر شمارش معکوس
2. سیستم پیشنهاد قیمت (Bidding) 
3. خرید فوری (Buy Now)
4. مدیریت فروشگاه‌های چندگانه
5. کیف پول داخلی
6. سیستم کمیسیون پویا
7. مدیریت دسته‌بندی سه سطحی
8. ویژگی‌های فنی محصولات
9. روش‌های ارسال متنوع
10. سیستم نظرات و امتیازدهی

## نقش‌های کاربری
- **Admin**: مدیر کل سیستم
- **Seller**: فروشنده
- **Buyer**: خریدار
- **Guest**: کاربر مهمان
