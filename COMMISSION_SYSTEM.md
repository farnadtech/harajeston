# سیستم سپرده و کمیسیون خودکار

## خلاصه
این سیستم امکان مدیریت خودکار سپرده شرکت در مزایده و کمیسیون سایت را فراهم می‌کند.

## قابلیت‌های اضافه شده

### 1. محاسبه خودکار سپرده
- **نوع محاسبه**: مبلغ ثابت یا درصد از قیمت پایه
- **قابل تنظیم توسط ادمین**: از طریق پنل مدیریت

### 2. کمیسیون سایت
- **نوع محاسبه**: مبلغ ثابت یا درصد از قیمت نهایی
- **پرداخت کننده**: خریدار، فروشنده، یا هر دو
- **تقسیم کمیسیون**: در صورت انتخاب "هر دو"، درصد تقسیم قابل تنظیم است

## فایل‌های ایجاد شده

### 1. Migration
- `database/migrations/2026_02_20_000001_create_site_settings_table.php`
  - جدول تنظیمات سایت با مقادیر پیش‌فرض

### 2. Models
- `app/Models/SiteSetting.php`
  - مدل تنظیمات با قابلیت کش

### 3. Services
- `app/Services/CommissionService.php`
  - محاسبه سپرده بر اساس تنظیمات
  - محاسبه کمیسیون بر اساس تنظیمات
  - کسر کمیسیون از طرفین
  - واریز به کیف پول سایت

### 4. Controllers
- `app/Http/Controllers/Admin/SettingsController.php`
  - مدیریت تنظیمات سپرده
  - مدیریت تنظیمات کمیسیون

### 5. Views
- `resources/views/admin/settings/index.blade.php`
  - صفحه تنظیمات سایت برای ادمین

### 6. Routes
- `GET /admin/settings` - نمایش صفحه تنظیمات
- `PUT /admin/settings/deposit` - به‌روزرسانی تنظیمات سپرده
- `PUT /admin/settings/commission` - به‌روزرسانی تنظیمات کمیسیون

### 7. Tests
- `tests/Feature/CommissionServiceTest.php`
  - تست محاسبه سپرده
  - تست محاسبه کمیسیون
  - تست کسر کمیسیون از طرفین

## تنظیمات پیش‌فرض

### سپرده
- نوع: درصد
- مبلغ ثابت: 1,000,000 تومان
- درصد: 10%

### کمیسیون
- نوع: درصد
- مبلغ ثابت: 50,000 تومان
- درصد: 5%
- پرداخت کننده: خریدار
- تقسیم (برای حالت "هر دو"): 50% خریدار، 50% فروشنده

## نحوه استفاده

### برای ادمین
1. وارد پنل مدیریت شوید
2. از منوی سمت راست، گزینه "تنظیمات سایت" را انتخاب کنید
3. تنظیمات سپرده و کمیسیون را به دلخواه تغییر دهید
4. تغییرات را ذخیره کنید

### برای توسعه‌دهندگان

#### محاسبه سپرده
```php
$commissionService = app(CommissionService::class);
$deposit = $commissionService->calculateDeposit($basePrice);
```

#### محاسبه کمیسیون
```php
$commission = $commissionService->calculateCommission($finalPrice);
```

#### کسر کمیسیون
```php
$result = $commissionService->deductCommission($listing, $buyer, $finalPrice);
// $result = [
//     'total_commission' => 50000,
//     'buyer_commission' => 30000,
//     'seller_commission' => 20000,
// ]
```

## تغییرات در سرویس‌های موجود

### AuctionService
- استفاده از `CommissionService` برای محاسبه سپرده
- کسر کمیسیون پس از پایان مزایده و پرداخت برنده
- کسر کمیسیون از خریدار و/یا فروشنده بر اساس تنظیمات

### DepositService
- استفاده از `CommissionService` برای محاسبه سپرده به جای مقدار ثابت

## کیف پول سایت
- کمیسیون‌ها به کیف پول کاربر با `user_id = 1` واریز می‌شود
- می‌توانید این رفتار را در متد `depositToSiteWallet` در `CommissionService` تغییر دهید

## نکات مهم
1. تمام تنظیمات در کش ذخیره می‌شوند (1 ساعت)
2. پس از تغییر تنظیمات، کش به صورت خودکار پاک می‌شود
3. کمیسیون فقط پس از پایان موفق مزایده و پرداخت برنده کسر می‌شود
4. در صورت انتخاب "هر دو" برای پرداخت کننده، می‌توانید درصد تقسیم را تنظیم کنید

## مثال‌های کاربردی

### مثال 1: سپرده 10% از قیمت پایه
```
قیمت پایه: 1,000,000 تومان
نوع سپرده: درصد
درصد سپرده: 10%
سپرده محاسبه شده: 100,000 تومان
```

### مثال 2: کمیسیون 5% از خریدار
```
قیمت نهایی: 2,000,000 تومان
نوع کمیسیون: درصد
درصد کمیسیون: 5%
پرداخت کننده: خریدار
کمیسیون خریدار: 100,000 تومان
کمیسیون فروشنده: 0 تومان
```

### مثال 3: کمیسیون 5% از هر دو (60% خریدار، 40% فروشنده)
```
قیمت نهایی: 2,000,000 تومان
نوع کمیسیون: درصد
درصد کمیسیون: 5%
پرداخت کننده: هر دو
تقسیم: 60% خریدار
کمیسیون کل: 100,000 تومان
کمیسیون خریدار: 60,000 تومان
کمیسیون فروشنده: 40,000 تومان
```
