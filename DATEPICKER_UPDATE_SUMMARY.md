## مشکلات احتمالی و راه‌حل

### مشکل: تقویم باز نمی‌شود
**راه‌حل:**
```bash
# مطمئن شوید فایل‌ها وجود دارند
ls public/js/persian-datepicker-package.js
ls public/css/persian-datepicker-package.css

# کش مرورگر را پاک کنید
Ctrl + Shift + R (Windows)
Cmd + Shift + R (Mac)
```

### مشکل: استایل‌ها اعمال نمی‌شوند
**راه‌حل:**
- کنسول مرورگر را باز کنید (F12)
- تب Network را بررسی کنید
- مطمئن شوید فایل CSS با موفقیت لود شده (200 OK)

### مشکل: تاریخ در فرمت اشتباه
**راه‌حل:**
- فرمت پیش‌فرض: `YYYY/MM/DD HH:mm`
- برای تغییر فرمت، options را تنظیم کنید
