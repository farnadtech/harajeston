تست فرم‌های احراز هویت

1. تست فرم ثبت نام

برای تست ثبت نام، فیلدهای زیر را پر کنید:

  • نام و نام خانوادگی: تست کاربر
  • ایمیل: test@example.com
  • رمز عبور: 12345678
  • تکرار رمز عبور: 12345678
  • قبول قوانین: بله

2. تست فرم ورود

برای تست ورود، از اطلاعات زیر استفاده کنید:

  • ایمیل: test@example.com
  • رمز عبور: 12345678

3. بررسی تغییرات

  • ✓ فرم ثبت نام حالا به روت POST /register متصل شده
  • ✓ فرم ورود حالا به روت POST /login متصل شده
  • ✓ تمام فیلدهای ثبت نام الزامی (required) هستند
  • ✓ فیلد تکرار رمز عبور اضافه شده
  • ✓ نمایش خطاها با استایل مناسب
  • ✓ قابلیت نمایش/مخفی کردن رمز عبور
  • ✓ حفظ مقادیر قبلی با old() در صورت خطا

4. فیلدهای الزامی در ثبت نام

  • نام و نام خانوادگی (name) - الزامی
  • ایمیل (email) - الزامی و باید یونیک باشد
  • رمز عبور (password) - الزامی، حداقل 8 کاراکتر
  • تکرار رمز عبور (password_confirmation) - الزامی
  • قبول قوانین (terms) - الزامی
