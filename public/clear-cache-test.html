🧪 تست بدون Cache

این صفحه فایل JS را بدون cache بارگذاری می‌کند

⚠️ اگر هنوز کار نمی‌کند:

Windows/Linux: Ctrl + Shift + Delete → Cache را پاک کنید

یا Hard Refresh: Ctrl + F5 یا Ctrl + Shift + R

Mac: Cmd + Shift + R

✅ چگونه تست کنیم:

  • روی فیلد اول کلیک کنید
  • سعی کنید تاریخ دیروز یا قبل‌تر را انتخاب کنید
  • روزهای گذشته باید خاکستری و غیرفعال باشند
  • نباید بتوانید روی آنها کلیک کنید
