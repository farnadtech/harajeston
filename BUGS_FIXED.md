# باگ‌های رفع شده

## 1. تصویر اصلی روی عکس‌های جدید در صفحه ویرایش ✅
- **مشکل**: روی تصاویر جدید نوشته "تصویر اصلی" نمایش داده میشد
- **راه حل**: تغییر label به "جدید" با رنگ سبز

## 2. جایگزینی عکس در صفحه ایجاد ✅
- **مشکل**: وقتی کاربر یک عکس انتخاب میکرد و بعد عکس دیگه، عکس جدید جایگزین قبلی میشد
- **راه حل**: 
  - نگه داشتن array از فایل‌های انتخاب شده
  - استفاده از DataTransfer API برای آپدیت input.files
  - اضافه کردن دکمه حذف (×) برای هر تصویر
  - تابع removeImage برای حذف تصاویر

## 3. تگ‌های قبلی در صفحه ویرایش نمایش داده نمیشن ⏳
- نیاز به بررسی

## 4. تصاویر فروشگاه 404 میدن ⏳  
- نیاز به بررسی مسیر تصاویر
