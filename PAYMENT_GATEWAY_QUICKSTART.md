## 🎯 مراحل استفاده (برای ادمین)

### گام 1: ورود به پنل ادمین
```
http://localhost/admin/payment-gateways
```

### گام 2: انتخاب درگاه
8 درگاه پرداخت آماده است:
- زرین‌پال (محبوب‌ترین)
- بانک ملت
- بانک سامان
- بانک پارسیان
- بانک پاسارگاد
- بانک صادرات
- بانک سپه
- ایران کیش

### گام 3: تنظیم درگاه
1. روی "ویرایش" کلیک کنید
2. اطلاعات احراز هویت را وارد کنید
3. وضعیت را روی "فعال" قرار دهید
4. ذخیره کنید

### مثال: تنظیم زرین‌پال
```
Merchant ID: xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx
```

## 💳 استفاده برای کاربران

### گام 1: ورود به کیف پول
```
http://localhost/wallet
```

### گام 2: شارژ کیف پول
1. مبلغ را وارد کنید (حداقل 10,000 تومان)
2. یکی از درگاه‌های فعال را انتخاب کنید
3. روی "پرداخت و شارژ کیف پول" کلیک کنید
4. پرداخت را در درگاه انجام دهید
5. پس از بازگشت، کیف پول شارژ می‌شود

### نکته: مالیات
مالیات 9% به صورت خودکار محاسبه و به مبلغ اضافه می‌شود.

## 📝 تنظیمات محیط (.env)

برای استفاده در production، اطلاعات درگاه‌ها را در `.env` اضافه کنید:

```env
# زرین‌پال
ZARINPAL_MERCHANT_ID=your-merchant-id

# بانک ملت
MELLAT_TERMINAL_ID=your-terminal-id
MELLAT_USERNAME=your-username
MELLAT_PASSWORD=your-password

# و غیره...
```

## 🔧 تنظیمات پیشرفته

### تغییر مالیات
در پنل ادمین > تنظیمات > کیف پول

### تغییر محدودیت‌ها
```php
// حداقل شارژ
SiteSetting::set('wallet_min_deposit', 10000);

// حداکثر شارژ
SiteSetting::set('wallet_max_deposit', 100000000);
```

## 🆘 عیب‌یابی

### مشکل: درگاه فعال نمی‌شود
✅ بررسی کنید اطلاعات احراز هویت صحیح باشد

### مشکل: پرداخت موفق نیست
✅ Callback URL را در تنظیمات درگاه ثبت کنید:
```
https://yourdomain.com/wallet/payment/callback
```

### مشکل: کیف پول شارژ نمی‌شود
✅ لاگ‌ها را بررسی کنید:
```bash
tail -f storage/logs/laravel.log
```
