# راهنمای مالیات شارژ کیف پول

## خلاصه
سیستم مالیات شارژ کیف پول به ادمین اجازه می‌دهد درصد مالیاتی را برای شارژ کیف پول تعیین کند که به صورت خودکار به مبلغ شارژ اضافه می‌شود.

## ویژگی‌ها

### 1. تنظیمات ادمین
- ادمین می‌تواند درصد مالیات را از صفحه تنظیمات تعیین کند
- مسیر: پنل ادمین → تنظیمات → تنظیمات کیف پول
- محدوده: 0% تا 100%
- پیش‌فرض: 0% (بدون مالیات)

### 2. محاسبه خودکار
وقتی کاربر مبلغ شارژ را وارد می‌کند:
- مبلغ مالیات = مبلغ شارژ × درصد مالیات ÷ 100
- مبلغ نهایی = مبلغ شارژ + مبلغ مالیات

**مثال:**
- مبلغ شارژ: 100,000 تومان
- درصد مالیات: 9%
- مبلغ مالیات: 9,000 تومان
- مبلغ قابل پرداخت: 109,000 تومان

### 3. نمایش برای کاربر
در فرم شارژ کیف پول، کاربر می‌بیند:
- مبلغ شارژ (مبلغی که وارد کرده)
- مالیات (با درصد)
- مبلغ قابل پرداخت (جمع کل)

این اطلاعات به صورت زنده (real-time) محاسبه و نمایش داده می‌شود.

## فایل‌های تغییر یافته

### 1. Backend

#### app/Services/WalletService.php
متد جدید اضافه شده:
```php
public function calculateChargeWithTax(float $amount): array
{
    $taxPercentage = \App\Models\SiteSetting::get('wallet_charge_tax', 0);
    $tax = ($amount * $taxPercentage) / 100;
    $total = $amount + $tax;
    
    return [
        'base_amount' => $amount,
        'tax' => $tax,
        'total' => $total,
        'tax_percentage' => $taxPercentage,
    ];
}
```

#### app/Http/Controllers/Admin/SettingsController.php
- اضافه شدن `wallet_charge_tax` به تنظیمات
- validation برای درصد مالیات (0-100)

- توضیحات نحوه محاسبه
- مثال عملی

#### resources/views/wallet/admin.blade.php

## نحوه استفاده

### برای ادمین:

1. وارد پنل ادمین شوید
2. به صفحه "تنظیمات" بروید
3. در بخش "تنظیمات کیف پول" درصد مالیات را وارد کنید
4. دکمه "ذخیره تنظیمات کیف پول" را بزنید

### برای کاربران:

1. وارد صفحه کیف پول شوید
2. در فرم "افزایش موجودی" مبلغ دلخواه را وارد کنید
3. اگر مالیات فعال باشد، جزئیات محاسبه نمایش داده می‌شود:
   - مبلغ شارژ
   - مالیات (با درصد)
   - مبلغ قابل پرداخت
4. دکمه "پرداخت" را بزنید

## نکات مهم

### 1. مبلغ ذخیره شده در کیف پول
- مبلغی که به کیف پول اضافه می‌شود = مبلغ شارژ (بدون مالیات)
- مالیات فقط در هنگام پرداخت دریافت می‌شود
- در تراکنش‌ها فقط مبلغ شارژ ثبت می‌شود

### 2. محدودیت‌ها
- حداقل و حداکثر مبلغ شارژ بر اساس مبلغ شارژ است (نه مبلغ نهایی)
- مالیات به محدودیت‌های min/max اضافه نمی‌شود

### 3. نمایش
- اگر درصد مالیات 0 باشد، بخش محاسبات نمایش داده نمی‌شود
- محاسبات به صورت زنده (با تایپ کردن) انجام می‌شود
- اعداد با فرمت فارسی نمایش داده می‌شوند

## تنظیمات پیشنهادی

### برای مالیات ارزش افزوده (VAT)
- ایران: 9%
- اروپا: معمولاً 19-25%
- آمریکا: بسته به ایالت 0-10%

### برای کارمزد پلتفرم
- کارمزد کم: 1-3%
- کارمزد متوسط: 3-5%
- کارمزد بالا: 5-10%

## مثال‌های عملی

### مثال 1: مالیات 9%
```
مبلغ شارژ: 100,000 تومان
مالیات (9%): 9,000 تومان
مبلغ قابل پرداخت: 109,000 تومان
```

### مثال 2: مالیات 5%
```
مبلغ شارژ: 500,000 تومان
مالیات (5%): 25,000 تومان
مبلغ قابل پرداخت: 525,000 تومان
```

### مثال 3: بدون مالیات
```
مبلغ شارژ: 200,000 تومان
مالیات (0%): 0 تومان
مبلغ قابل پرداخت: 200,000 تومان
```
