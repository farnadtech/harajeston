# تغییرات سیستم ویرایش آگهی‌ها

## تغییرات اعمال شده

### 1. سیستم Pending Changes برای آگهی‌های فعال
- **مشکل**: وقتی آگهی فعال ویرایش می‌شد، بلافاصله تغییرات اعمال می‌شد
- **راه‌حل**: 
  - ایجاد جدول `listing_pending_changes` برای ذخیره تغییرات پیشنهادی
  - تغییرات در یک نسخه جداگانه ذخیره می‌شوند
  - بعد از تایید ادمین، تغییرات روی آگهی اصلی اعمال می‌شوند
  - آگهی اصلی بدون تغییر باقی می‌ماند تا تایید شود

### 2. محدودیت ویرایش برای آگهی‌های با پیشنهاد فعال
- **قانون**: اگر آگهی دارای پیشنهاد فعال باشد، فقط این فیلدها قابل ویرایش هستند:
  - توضیحات (description)
  - روش‌های ارسال (shipping methods)
- **فیلدهای غیرقابل ویرایش**:
  - عنوان
  - دسته‌بندی
  - وضعیت کالا
  - قیمت شروع
  - قیمت خرید فوری
  - زمان شروع و پایان
  - مشخصات فنی

### 3. حذف فیلد bid_increment از فرم‌ها
- **تغییر**: فیلد "حداقل افزایش پیشنهاد" از فرم‌های ایجاد و ویرایش حذف شد
- **جایگزین**: تنظیم `default_bid_increment` در پنل ادمین
- **مکان**: تنظیمات سایت > تنظیمات آگهی‌ها > گام افزایش پیشنهاد پیش‌فرض
- **مقدار پیش‌فرض**: 10,000 تومان

## ورک فلو جدید

### ویرایش آگهی بدون پیشنهاد
1. فروشنده فرم ویرایش را پر می‌کند
2. اگر تنظیم تایید فعال باشد:
   - تغییرات در جدول `listing_pending_changes` ذخیره می‌شوند
   - آگهی اصلی بدون تغییر باقی می‌ماند
   - پیام: "تغییرات شما ثبت شد و برای تایید ادمین ارسال گردید"
3. اگر تنظیم تایید غیرفعال باشد:
   - تغییرات مستقیماً روی آگهی اعمال می‌شوند
   - پیام: "آگهی با موفقیت به‌روزرسانی شد"

### ویرایش آگهی با پیشنهاد فعال
1. فروشنده فرم ویرایش را باز می‌کند
2. نوتیفیکیشن زرد نمایش داده می‌شود: "این آگهی دارای پیشنهاد فعال است"
3. فقط فیلدهای توضیحات و روش‌های ارسال فعال هستند
4. سایر فیلدها disabled و خاکستری هستند
5. پس از ذخیره:
   - فقط توضیحات و روش‌های ارسال به‌روزرسانی می‌شوند
   - پیام: "فقط توضیحات و روش‌های ارسال به‌روزرسانی شدند"

### تایید تغییرات توسط ادمین
1. ادمین به صفحه مدیریت آگهی می‌رود
2. اگر pending changes وجود داشته باشد، دکمه "مشاهده تغییرات" نمایش داده می‌شود
3. ادمین تغییرات را بررسی می‌کند
4. ادمین می‌تواند:
   - تایید کند: تغییرات روی آگهی اصلی اعمال می‌شوند
   - رد کند: تغییرات حذف می‌شوند و آگهی بدون تغییر باقی می‌ماند

## تنظیمات جدید

### گام افزایش پیشنهاد پیش‌فرض
- **مکان**: پنل ادمین > تنظیمات > تنظیمات آگهی‌ها
- **نام فیلد**: `default_bid_increment`
- **مقدار پیش‌فرض**: 10,000 تومان
- **حداقل**: 1,000 تومان
- **کاربرد**: برای تمام آگهی‌های جدید استفاده می‌شود

## نکات مهم

1. **آگهی‌های قدیمی**: آگهی‌هایی که قبل از این تغییرات ایجاد شده‌اند، همچنان bid_increment خودشان را دارند
2. **ادمین**: ادمین می‌تواند بدون محدودیت آگهی‌ها را ویرایش کند
3. **Pending Changes**: فقط برای آگهی‌های active و pending اعمال می‌شود
4. **محدودیت ویرایش**: فقط برای آگهی‌هایی که حداقل یک پیشنهاد دارند اعمال می‌شود
